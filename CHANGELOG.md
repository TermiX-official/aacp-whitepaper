# Changelog

All notable changes to the AACP whitepaper will be documented in this file.

## [0.1.0] - 2026-03-20

### Added
- Initial whitepaper release
- Protocol overview: four roles (Client, Provider, Evaluator, Arbitrator)
- Agent identity via ERC-8004 with capability declaration and reputation scoring
- Job lifecycle extending ERC-8183 with staking checkpoints and dispute states
- Four-level verification system (Manual, TEE, zkVM, TEE+zkVM)
- Economic model: fee structure, staking requirements, reward mechanisms, slashing conditions
- Protocol revenue model: on-chain fees, SaaS subscriptions, Prover-as-a-Service, TEE hosting
- Dispute resolution with VRF arbitrator selection and commit-reveal voting
- Anti-gaming measures: Sybil resistance, collusion detection, verification program integrity
- Deployment plan for Base, BSC, and Ethereum
- Four-phase roadmap (Q2 2026 - Q1 2027)
- Security considerations for smart contracts, TEE, zkVM, and economic attacks
